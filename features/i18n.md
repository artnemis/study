# Internationalization (i18n)

## Overview

Artnemis supports 5 languages: Italian (it), English (en), French (fr), Spanish (es), German (de).

## Architecture

- Client-side React Context (`LocaleProvider`) wraps the entire app
- Dictionary files in `lib/i18n/dictionaries/` — one TypeScript file per locale
- `useT()` hook returns the current dictionary
- `useLocale()` hook returns `{ locale, setLocale, t }`
- Locale is persisted in a cookie (`locale`) and in the user's profile settings

## Dictionary Structure

All dictionaries implement the `Dictionary` interface (`lib/i18n/index.ts`):
- Navigation labels
- Landing page content
- Dashboard labels
- Module CRUD labels
- Quiz and plan generation labels
- Profile page labels
- Auth page labels
- Common labels (loading, error, retry)

## Adding a New Language

1. Create `lib/i18n/dictionaries/{code}.ts` implementing `Dictionary`
2. Add the locale code to `LOCALES` in `lib/i18n/index.ts`
3. Import and register in `lib/i18n/context.tsx` dictionaries map
4. Add label to `LOCALE_LABELS` in `app/profile/page.tsx`
